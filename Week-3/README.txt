#dice Game Rules:
- Every Player will get change to roll a dice.
- Game will start from random player and continue in circle.
- Who gets the bigger number, they're the winner.
- Press "Y/y" to roll the dice for the current player.
- Press "N/n" to finish the game.

# Project Structure

The project files and folders are organized as follows:
